## Getting Started

### Installation

A. Develop apps:
  1. Clone the repo
     ```sh
     git clone https://github.com/rwndy/phonebook-project.git
     ```
  2. Install packages
     ```sh
     yarn install
     ```
  3. Start react
     ```sh
     yarn start 
     ```
